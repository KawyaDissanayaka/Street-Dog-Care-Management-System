A **full-stack application** to manage street dog care, with a **Spring Boot backend** and **React frontend**.

---

## 🌟 Features

* Track street dogs and their health records
* Add, edit, and delete dog information
* Interactive frontend built with React
* Backend REST API built with Spring Boot

---

## ⚡ Tech Stack

* **Backend:** Java 17, Spring Boot, Maven
* **Frontend:** React, Node.js, npm
* **Database:** (Future) MySQL / H2 / PostgreSQL

---

## 🚀 Getting Started

### Backend

```bash
cd Street-Dog-Care-Management-System
./mvnw spring-boot:run
```

> Backend runs at `http://localhost:8080`
> *Note:* Currently requires database configuration to fully start.

### Frontend

```bash
cd street-dog-frontend
npm install
npm start
```

> Frontend runs at `http://localhost:3000`
